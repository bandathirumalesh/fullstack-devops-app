College Student Registration System

Build a simple website with login, users, and student registration.

This web application includes frontend pages, backend logic, and a database so you can learn how a complete student portal works.

Login or create a new user

Use the right-side forms to create an account and sign in.

Register college students

After login, you can submit student details and save them to the database.

Open in IntelliJ easily

This is a standard Spring Boot Maven project, so it imports cleanly into IntelliJ IDEA.
